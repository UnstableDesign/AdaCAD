
## Parameters
- `ends`: shifts the position of the ends in draft b relative to a
- `pics`: shifts the position of the pics in draft b relative to a
- `bitfield function`: Maths expression that uses x/y values to return a boolean value for each cell, to make 'bitfield' patterns,


## Application
No established application, but a fun way to see how to apply math makes interesting drafts.
